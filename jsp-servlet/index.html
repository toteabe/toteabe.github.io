SDKMAN

Comenzaremos con la instalación de SDKMAN en bash.

Se trata de una herramienta para la gestión paralela de múltiples SDK, llamados “candidatos”. Proporciona una interfaz de línea de comandos (CLI) y API para enumerar, instalar, cambiar y eliminar candidatos.

Además, se encarga de configurar las variables de entorno por nosotros. Permite a los desarrolladores instalar SDK basados ​​en JVM como Java, Groovy, Scala, Kotlin y Ceylon. También se admiten Maven, Gradle, SBT, Spring Boot, Vert.x y muchos otros.

Es una utilidad gratuita, liviana y de código abierto escrita en Bash.

Para instalarlo en un sistema linux utilizaríamos instalación

curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
sdk version

Seguidamente, listaremos todas las opciones de SDK disponibles para Java mediante:

sdk list java
